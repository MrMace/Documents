# 📚 Documents

A collection of technical notes, setup guides, and reference material.

## Structure

```

/docs/ <category>/ <topic>.md

```

- Files are grouped by category
- Each file should cover a single topic

## Usage

Browse the `/docs` folder to find what you need.

## Notes

- Keep filenames simple and consistent (e.g. `cli-setup.md`)
- Avoid committing secrets or credentials
- Update docs as things change
